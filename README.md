# ESP32-C3 Simon Says Game

This project is a Simon Says memory game developed using the ESP32-C3 microcontroller.

The system generates a sequence of lights and sounds that the player must memorize and repeat correctly. As the game progresses, the sequence becomes longer and more challenging.

## Features

- Random sequence generation
- Increasing difficulty levels
- LED and sound feedback
- User input validation
- Score progression system
- Real-time game interaction

## Components

- ESP32-C3 Development Board
- LEDs
- Push Buttons
- Buzzer
- Breadboard

## Technologies
